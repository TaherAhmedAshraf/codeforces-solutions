# Codeforces Solutions

This repo is where I keep all my Codeforces problem solutions.

Main goal is simple:

## 📂 Structure

I’m keeping things simple for now.

```
codeforces-solutions/
│
├── rating/
│   ├── 800/
│   ├── 1000/
│   ├── 1200/
│   └── ...
│
├── contests/
│   └── Round-XXX/
│
└── templates/
```

Most problems are organized by rating.
Contest folders are just for quick access when I solve during a round.
